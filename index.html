# 字符 标题

📋 使用说明

表格格式
国际站热词导出文件（.xls）
A列=关键词，B列=搜索指数
无需整理，直接上传原始文件
词根识别
自动识别搜索指数最高、
覆盖率最广的核心词组作为词根
词根固定出现在每条标题最右边
组合规则
• 词根固定最右
• 按指数高→低从左→右排列
• 同标题内单词不重复
• 每条标题 ≤128字符，尽量填满
颜色说明
🟢 绿色：字符数 ≥ 120（优质）
🟡 黄色：字符数 < 100（偏短）
